## Facebook 行銷小工具

### 功能
+ 抓取貼文按讚的 fb_id。
+ 抓取貼文留言的 fb_id。
+ 抓取貼文分享的 fb_id。
+ 抽獎，從 fb_id 名單隨機選取。
+ 匯出 CSV 檔案

註：(需要得到使用者的 Facebook 帳戶授權)

### 技術
+ [Jquery](https://github.com/jquery/jquery)
+ [Bootstrap](https://github.com/twbs/bootstrap)
+ [tableExport.jquery.plugin](https://github.com/kayalshri/tableExport.jquery.plugin)
